## 管理输出及插件

-----

- 多个文件输入输出
```javascript
const path = require('path');

module.exports={
    entry:{
        app:'./src/app.js',
        print:'./src/print.js'
    },
    output:{
        filename:'[name].bundle.js',
        path:path.resolve(__dirname,'dist')
    }    
}
//可以看到，webpack 生成 print.bundle.js 和 app.bundle.js 文件
```
- plugin

```text
html-webpack-plugin
安装 npm i -D html-webpack-plugin
使用如下
```
```javascript
const path = require('path');
const HtmlWebpackPlugin = require('html-')

module.exports={
    entry:'index.js',
    output:{
        filename:'bundle.js',
        path:path.resolve(__dirname,'./dist')
    },
    plugins:[
        new HtmlWebpackPlugin({
            title:'this is a file generated by HtmlWebpackPlugin'
        })
    ]
}
```

- clean-webpack-plugin
```text
通常，在每次构建前清理 /dist 文件夹，是比较推荐的做法，因此只会生成用到的文件。
让我们完成这个需求。
```
```javascript
const CleanWebpackPlugin = require('clean-webpack-plugin');
plugins: [
+     new CleanWebpackPlugin(['dist']),
      new HtmlWebpackPlugin({
        title: 'Output Management'
      })
    ],
```
